# HeadFirstDesign
设计模式代码，本质是将代码中经常变化的代码提取出来，以提高扩展性、可维护性。

- Strategy 策略模式 , 委托（delegate）
- Observer 观察者模式，建立一对多依赖，更新时会收到通知。
- Decorator 装饰者模式
- Abstract Factory
- Factory Method  工厂模式
- Singleton 单件模式
- Command
- Adapter
- Facade
- Template Method
- Iterator
- Composite
- State
- Proxy

## 设计原则 
- 针对接口编程
